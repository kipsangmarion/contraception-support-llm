# Results Plots

This directory contains visualizations of experimental results.

## Files

- `accuracy_comparison.png` - Bar plot comparing accuracy across experiments
- `confusion_matrix.png` - Error type confusion matrix
- `rl_reward_curves.png` - Reinforcement learning reward trends
- `memory_recall_accuracy.png` - Memory test results across conditions
- `latency_comparison.png` - Response latency box plots
- `error_distribution.png` - Distribution of error types
