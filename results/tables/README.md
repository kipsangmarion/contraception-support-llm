# Results Tables

This directory contains CSV files with experimental results.

## Files

- `exp1_baseline.csv` - Baseline LLM knowledge test results
- `exp2_anchored.csv` - Anchored prompt evaluation results
- `exp3_rag_comparison.csv` - RAG vs non-RAG comparison
- `exp4a_long_session.csv` - Long-session forgetting test results
- `exp4b_multi_session.csv` - Multi-session memory test results
- `exp5_adherence_rl.csv` - Adherence RL experiment results
- `statistical_tests.csv` - Statistical significance tests
- `dataset_statistics.csv` - Dataset statistics and metadata
